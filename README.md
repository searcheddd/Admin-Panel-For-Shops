# YourShop Admin Panel

A modern admin dashboard for e-commerce stores.

## Features
- Product Management
- Categories System
- Offers & Discounts
- Customer Reviews
- Store Configuration
- Dashboard Analytics
- Promotional Popups
- LocalStorage Database
